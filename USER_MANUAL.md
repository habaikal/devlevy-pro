# DevLevy Pro 사용자 매뉴얼

**DevLevy Pro**는 복잡한 개발부담금을 쉽고 빠르게 산정해 주는 전문가용 웹 애플리케이션입니다. 브라우저 기반으로 동작하여 별도의 설치 없이 어디서든 접근 및 사용이 가능합니다.

---

## 🚀 시작하기 전에: 상단 툴바(메뉴) 알아보기
우측 상단에 위치한 아이콘 버튼들을 통해 앱의 주요 기능들을 제어할 수 있습니다.

- ➕ **새 프로젝트(초기화)**: 모든 입력값을 초기화하고 완전히 새로운 계산을 시작합니다.
- 📁 **프로젝트 관리**: 현재 입력된 상태를 이름을 지정하여 저장하거나, 이전에 저장했던 프로젝트 목록을 관리/불러옵니다.
- 🔄 **불러오기(임시)**: 브라우저에 임시 보관된(마지막에 수동 저장한) 단일 데이터를 가져옵니다.
- 💾 **저장하기(임시)**: 현재 진행 상태를 브라우저에 임시로 저장하여, 창을 실수로 닫아도 복구할 수 있게 합니다.
- 📤 **엑셀 가져오기**: 기존에 전용 포맷으로 내보냈던 엑셀 파일(.xlsx)을 불러와 모든 사업 변수 데이터를 자동 입력합니다.
- 🖨️ **프린트 및 PDF**: 계산 결과를 불필요한 테두리 없이 깔끔한 형태의 A4 스마트 보고서로 출력하거나 PDF로 저장합니다.
- ⚙️ **환경 설정**: 정밀 분석 팝업을 띄우기 위한 Google Gemini AI API 키를 내 브라우저에 안전하게 등록합니다.
- 📥 **보고서 다운로드**: 현재 사업 요약 및 항목들을 엑셀(.xlsx) 파일로 로컬 PC에 즉시 다운로드 합니다.

---

## 📝 1단계: 사업 기본정보 입력 (필수)
가장 기본적인 사업 개요를 최우선으로 입력하는 단계입니다.
1. 왼쪽 내비게이션 메뉴 영역에서 **[1. 사업 기본정보]** 탭을 클릭하세요.
2. **사업면적(㎡)** 칸에 대상 면적을 숫자로 입력합니다. (예: 1500)
3. **개발시작일**과 **개발종료일**을 달력 아이콘을 눌러 각각 선택합니다. (종료일은 시작일보다 빠를 수 없도록 방어 처리되어 있습니다.)
4. (선택형) 이어지는 2단계에서 지가 API 조회를 자동으로 해보고 싶다면 **공공데이터포털 서비스키**와 대상지의 **PNU 고유코드(19자리)**를 추가로 기입합니다.

---

## 🌍 2단계: 개시/종료 지가 및 가중 변수 평가
납부할 세금의 기초가 되는 지가상승분과 추가 평가 변수들을 입력하여 정밀한 사업 타당성을 구성합니다.
1. **[2. 지가 및 변수]** 탭으로 이동합니다.
2. **개시시점지가(시작)** 카드 영역:
   - 해당 [연도]를 빈칸에 넣고 **[API 조회]** 버튼을 누르면, 1단계에서 입력한 PNU 번호 기반으로 국토부 공시지가가 자동 기입됩니다.
   - API를 이용하지 않고 **수기**로 직접 개시시점 ㎡당 단가를 통금액으로 작성해도 무방합니다.
3. **종료시점지가(종료)** 카드 영역:
   - 개시시점과 동일하게 종료 연도 지정 및 지가를 수동 또는 자동으로 세팅합니다.
4. **정상지가상승분(원)** 카드 영역:
   - **[자동 계산(표준상승률)]** 버튼을 누르면 시작일과 종료일 기간(연도 단위)을 계산해, 지가 상승분(기본 5% 복리 기준)이 공제 금액으로 자동 세팅됩니다.
5. **32개 지가 산정 변수** (심화):
   - 용도지역, 지형지세, 도로조건 등 32가지 변수를 현황에 맞게 체크하시면 [변수 기반 자동 산정] 활용 시 단가 가중치에 직접 반영됩니다.

---

## 💰 3단계: 개발비용 산출명세서
부담금 계산 시 차감되어 실제 세액을 큰 폭으로 줄여주는 공제 비용 항목들을 등록합니다.
1. **[3. 개발비용]** 탭을 클릭하여 페이지를 엽니다.
2. 프로젝트에 청구된 비용(단위: 원)을 각 항목에 맞춰 기재합니다.
   - 순공사비, 조사비, 설계비, 일반관리비, 기부채납액, 기타비용
3. 입력할 때 마다 **'총 개발비용 합계'** 박스에 실시간으로 집계되며, 하단의 원형 도넛 차트(Pie Chart)를 통해 비용이 어느 항목에 가장 큰 비중으로 지출되었는지 시각적으로 분석할 수 있습니다.

---

## 📊 4단계: 최종 산정 결과 확인 및 리포팅
모두 작성된 정보를 바탕으로 최종 개발이익과 납부해야 할 세금 규모를 확인하는 마지막 단계입니다.
1. **[4. 산정 결과]** 탭으로 전환합니다.
2. 상단 요약 카드에 총 개발비용, 개발이익(지가차액-비용), 그리고 **최종 부과 대상 개발부담금(통상 25%)**의 절대값이 크게 나타납니다.
3. 하단에는 비용, 이익, 공제 부담금 비중을 한눈에 비교하는 막대 그래프가 렌더링 됩니다.

**[🌟 핵심 보너스 기능: AI 전문가 분석]**
- 우측 상단 ⚙️(환경 설정)에서 Google AI Studio에서 무료로 발급받은 **Gemini API Key**를 입력해둔 경우 [지금 분석 실행하기] 버튼을 누를 수 있습니다.
- 인공지능이 입력하신 지역, 사업 규모, 공사비 규모 간의 상대적 위험도나 특이 사항을 감정평가사 시각에서 팝업(Modal)으로 짚어드려 더욱 신뢰성 있는 문서 작성이 가능합니다.

---

> **💡 핵심 꿀팁**: 여러 개의 사업(필지)을 동시에 벤치마킹 중이시라면, 하나의 프로젝트 지표 입력을 끝마칠 때마다 상단 내비게이션 바의 **[📁 프로젝트 관리]** 버튼을 눌러 **'이름을 정해서 저장'** 하시기 바랍니다. 이렇게 하면 여러 개의 시뮬레이션을 언제든 손쉽게 스왑(초기화/불러오기)하며 비교 평가할 수 있습니다!
