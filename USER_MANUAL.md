# DevLevy Pro 사용자 매뉴얼

**DevLevy Pro**는 복잡한 개발부담금을 쉽고 빠르게 산정해 주는 전문가용 웹 애플리케이션입니다. 브라우저 기반으로 동작하여 별도의 설치 없이 스마트폰이나 PC 어디서든 접속 및 사용이 가능합니다.

---

## 🚀 0단계: 상단 툴바(메뉴) 버튼 알아보기
우측 상단에 위치한 원형 및 막대형 아이콘 버튼들을 통해 앱의 주요 기능들을 제어할 수 있습니다.

- ➕ **새 프로젝트(초기화)**: 모든 입력값을 초기화 상태로 되돌리고, 새로운 부지를 대상으로 계산을 시작합니다.
- 📁 **프로젝트 관리**: 현재 입력된 모든 숫자와 상태를 이름을 지정하여 저장하거나, 이전에 저장했던 프로젝트 목록들을 다시 불러옵니다.
- 🔄 **불러오기(임시)**: 브라우저에 임시로 보관된 내역 단위의 단일 데이터를 가져옵니다.
- 💾 **저장하기(임시)**: 현재 진행 상태를 브라우저에 임시 저장하여, 창을 실수로 닫아도 복구할 수 있게 합니다.
- 📤 **엑셀 가져오기**: 기존에 전용 포맷으로 내보냈던 엑셀 파일(.xlsx)을 불러와 모든 지번 데이터를 화면에 자동 세팅합니다.
- 🖨️ **프린트 및 PDF**: 계산 결과를 불필요한 레이아웃(좌우측 버튼) 없이 깔끔한 형태의 A4 스마트 보고서로 출력하거나 PDF로 저장합니다. 
- ⚙️ **환경 설정**: 정밀 분석 팝업을 띄우기 위한 Google Gemini AI API 키를 내 브라우저에 접속 시 안전하게 1회성(또는 지속) 암호화 등록합니다.

---

## 📝 1단계: 사업 기본정보 입력 (필수)
가장 핵심이 되는 사업 개요를 최우선으로 입력하는 단계입니다.
1. 왼쪽 내비게이션 메뉴 영역에서 **[1. 사업 기본정보]** 탭을 클릭하세요.
2. **사업면적(㎡)** 칸에 대상 토지 면적을 숫자로 입력합니다. (예: 1500)
3. **개발시작일**과 **개발종료일**을 달력 아이콘을 눌러 각각 선택합니다. (주의: 종료일은 시작일보다 빠를 수 없습니다.)
4. (선택형) 다음 단계에서 지가 API 조회를 자동으로 실행하려면 **공공데이터포털 서비스키**와 대상지의 **PNU 고유코드(19자리 숫자)**를 추가로 기입합니다.

---

## 🌍 2단계: 개시/종료 지가 및 가중 변수 평가
납부할 세금의 기초가 되는 정상지가 상승분과, 개발 요인에 따른 추가 평가 변수들을 입력하여 정밀한 사업 타당성을 구성합니다.
1. **[2. 지가 및 변수]** 탭으로 이동합니다.
2. **개시시점지가(시작)** 영역:
   - 해당 연도를 빈칸에 넣고 **[API 조회]** 버튼을 누르면, 1단계에서 입력한 PNU 번호를 기반으로 국토부 공시지가가 자동 입력됩니다. 
   - 자동입력 API를 이용하지 않고 **수기로 직접** 개시시점 ㎡당 단가를 입력해도 무방합니다.
3. **종료시점지가(종료)** 영역:
   - 개시시점과 동일하게 종료 연도 지정 및 지가를 수동 또는 자동으로 세팅합니다.
4. **정상지가상승분(원)** 영역:
   - **[자동 계산(표준상승률)]** 버튼을 누르면 시작일과 종료일 기간에 비례해, 지가 상승분(기본 5% 복리 기준)이 공제 금액으로 자동 산출됩니다.
5. **32개 지가 변수 체크** (심화 항목):
   - 용도지역, 지형지세, 도로조건 등 32가지 변수를 현장에 맞게 각각 체크하시면, 자동 산정 활용 시 토지 단가 가중치에 디테일하게 반영됩니다.

---

## 💰 3단계: 개발비용 산출명세서
부담금 계산 시 차감되어야 하는(실제 세액을 줄여주는) 공제 비용 항목들을 등록합니다.
1. **[3. 개발비용]** 탭을 클릭하여 페이지를 엽니다.
2. 프로젝트 진행에 지출(예정)된 비용(단위: 원)을 각 항목에 맞춰 기재합니다.
   - 대상: 순공사비, 조사비, 설계비, 일반관리비, 기부채납액, 기타비용
3. 입력할 때마다 화면상의 **'총 개발비용 합계'**가 실시간으로 집계되며, 하단의 원형 도넛 차트(Pie Chart)를 통해 비용이 어느 항목에 가장 크게 지출되었는지 직관적으로 시각 분석할 수 있습니다.

---

## 📊 4단계: 최종 산정 결과 확인 및 리포팅
모두 작성 완료된 정보를 바탕으로 최종 개발 사업 수익과, 지자체에 최종 납부해야 할 세금 규모를 뽑아보는 마지막 단계입니다.
1. **[4. 산정 결과]** 탭으로 전환합니다.
2. 우측 상단 요약 카드에 총 개발비용, 개발이익(총 지가 상승차액 - 투입비용), 그리고 **최종 부과 대상 개발부담금(통상 25% 가율 적용)** 의 절대값이 큰 숫자로 나타납니다.
3. 하단에는 비용, 누적 이익, 공제 부담금 비중 등을 한눈에 비교하는 요약형 막대 그래프 및 테이블이 제공됩니다.
4. 내역을 저장하고 싶다면, 우측 상단의 📥 **보고서 다운로드**를 누르거나 🖨️ **[프린트] 버튼**을 눌러 A4 사이즈의 PDF 백업용 문서로 전환하세요!

> **💡 핵심 이용 꿀팁: AI 분석과 다중 필지 관리**
> - **AI 전문가 팝업**: 우측 상단 ⚙️(설정)에서 무료 발급받은 **Gemini API Key**를 한 번만 등록해두시면 결과 창에서 언제든 [분석 실행] 버튼을 눌러보실 수 있습니다. 인공지능이 입력하신 데이터의 공사비 밸런스나 리스크를 전문가 시각에서 팝업으로 조언해 줍니다.
> - **다중 비교 평가**: 여러 개의 필지 사업성을 동시에 분석 중이라면, 하나의 입력이 끝날 때마다 상단 **[📁 프로젝트 관리]** 아이콘을 눌러 꼭 **'이름을 정해서 별도 저장'** 해두세요! 언제든 클릭 한 번에 저장된 각기 다른 사업지 데이터 뷰로 빠르게 전환할 수 있습니다.
